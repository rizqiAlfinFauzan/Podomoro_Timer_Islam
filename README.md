# 🍅 Pomodoro — Academic Focus Timer

A feature-rich, beautifully designed Pomodoro timer tailored for focus and spiritual mindfulness. This application is contained entirely within a single HTML file and seamlessly integrates productivity tools with Islamic daily routines, such as Al-Quran playback and automated prayer time alerts.

## ✨ Key Features

* **Customizable Pomodoro Timer:** Set your preferred Study Time, Break Time, and Number of Cycles.
* **Integrated Al-Quran Player:** Listen to various Qoris directly within the app while studying. Automatically plays the next Ayat.
* **Layered Prayer Alerts:** Never miss a prayer. Features a 3-layer reminder system:
    * *Pre-reminder:* Gentle notification 10 minutes before Adzan.
    * *Adzan Alert:* Automatically pauses your timer and displays a prominent visual/audio alert when prayer time arrives.
    * *Iqamah Reminder:* Gentle nudge 5 minutes after Adzan.
* **Mindful Breaks:** Features a Box Breathing animation and a relaxing "Cat Walking" animation to help you decompress during break intervals.
* **Picture-in-Picture (PiP):** Keep the timer visible as a floating window above your other tabs and applications.
* **Custom Themes:** Choose from 5 elegant, distraction-free color palettes (Classic Crimson, Midnight Navy, Forest Teal, Warm Amber, Deep Forest).
* **Session Tracking & Dzikir:** Tracks your focus statistics over the last 7 days and provides a post-session Dzikir counter.

## 🚀 How to Use

This project requires no installation, dependencies, or backend server. 

1. Download or clone this repository.
2. Open the `.html` file directly in any modern web browser (Chrome, Firefox, Edge, Safari).
3. **Location Permission:** Upon first load, the app will ask for location access to provide accurate prayer times. You can grant access, enter your city manually, or use the default location.

## ⌨️ Keyboard Shortcuts

* `Space` : Play / Pause Timer
* `R` : Reset Session
* `M` : Mute / Unmute Al-Quran Audio
* `?` : Toggle Shortcuts Help Panel
